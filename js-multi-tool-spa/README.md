# Proyecto JS SPA - 20 Herramientas e Interacciones

Una **Single Page Application (SPA)** completa desarrollada con JavaScript Vanilla, que cuenta con 20 herramientas y ejercicios interactivos diferentes.

## 🚀 Descripción General

Este proyecto integra 20 funcionalidades distintas en una interfaz única y fluida. Desde una calculadora básica hasta un tablero Kanban dinámico, cada ejercicio demuestra desde manipulación básica del DOM hasta lógica avanzada utilizando JavaScript puro.

## 🛠️ Herramientas Incluidas

| # | Funcionalidad | Descripción |
|---|---|---|
| 01 | **Calculadora** | Operaciones aritméticas básicas. |
| 02 | **Conversor de Temp** | Conversión entre Celsius, Fahrenheit y Kelvin. |

| 08 | **Generador de Pass** | Generador de contraseñas seguras y personalizables. |

| 10 | **Conversor de Divisas** | Tasas de cambio para múltiples monedas. |
| 11 | **Validación de Form** | Lógica robusta de validación en el lado del cliente. |
| 12 | **Adivina el Número** | Juego de lógica interactivo con sistema de intentos. |
| 13 | **Cronómetro** | Temporizador de precisión con función de vueltas. |

## 💻 Aspectos Técnicos

- **JavaScript Vanilla**: No se utilizaron frameworks ni librerías externas para la lógica.
- **LocalStorage**: Persistencia de ajustes de tema y estado de la aplicación.
- **CSS Moderno**: Uso de Flexbox y CSS Grid para una experiencia responsiva.
- **API Drag & Drop**: Implementación nativa del navegador para el tablero Kanban.
- **API Intl**: Formateo especializado de monedas y números.

## 📖 Cómo Usar

1. Clona este repositorio.
2. Abre el archivo `index.html` en cualquier navegador moderno.
3. Explora los ejercicios utilizando la barra de navegación superior.
